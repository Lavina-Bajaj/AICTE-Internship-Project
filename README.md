# 📘 AI Study Buddy

AI-powered learning assistant built using Python, Streamlit, and Hugging Face Llama AI.

## 🚀 Features
- Explain study topics in simple language
- Summarize notes instantly
- Generate exam-ready answers
- Create viva questions
- Sidebar history of previous questions

## 🛠 Technologies
- Python
- Streamlit
- Hugging Face API (Llama 3)

## ▶️ Run Locally
pip install -r requirements.txt
streamlit run study_buddy_app.py
